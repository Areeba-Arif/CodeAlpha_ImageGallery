# 🖼️ CodeAlpha Image Gallery

A modern and responsive **Image Gallery** developed as part of my **CodeAlpha Frontend Development Internship** using **HTML, CSS, and JavaScript**.

The project provides an interactive gallery experience with image categorization, filtering, lightbox preview, and next/previous image navigation in a clean and user-friendly interface.

---

# 📌 Project Overview

This project allows users to browse a collection of images organized into different categories. Users can filter images based on their interests and preview them in a fullscreen lightbox with smooth navigation controls.

The gallery is fully responsive, making it suitable for desktops, tablets, and mobile devices.

---

# ✨ Features

* 🏠 Landing (Home) page
* 🖼️ Interactive Image Gallery
* 🔍 Category-based image filtering
* 🎨 Categories:

  * Color Photography
  * Black & White Photography
  * Architecture
  * Abstract Art
* 💡 Lightbox image preview
* ⬅️ Previous image navigation
* ➡️ Next image navigation
* ❌ Close lightbox by button or outside click
* 📱 Fully responsive layout
* ✨ Smooth hover effects and modern UI
